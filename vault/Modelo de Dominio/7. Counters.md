## 7. Counters


**Fuente:** Counters

### ICounterData
**Tipo:** Entidad — `id` identifica una instancia con estado propio (valor actual del contador) que persiste en los datos de guardado del jugador y puede mutar con el tiempo. No es un catálogo compartido ni un agrupador de valores intercambiable.

```
ICounterData: {
  id: string,              // requerido — debe ser único, persiste en datos de guardado
  name: string,            // requerido — nombre de display
  default_value?: number,  // default 0
  min?: number,            // default -MAX_INT
  max?: number             // default MAX_INT
}
```
