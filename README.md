# LCP Builder

Aplicación Flutter/Dart para crear y editar archivos `.lcp` (paquetes de contenido compatibles con COMP/CON, TTRPG Lancer) mediante formularios visuales.

Proyecto personal usado como vehículo de aprendizaje de arquitectura de software (Clean/Hexagonal Architecture) y pieza de portfolio. Ver `CLAUDE.md` para el contexto completo de mentoría y `vault/00 - Inicio.md` para el estado del proyecto.

## Estructura

- **`app/`** — código Flutter/Dart (dominio, adapters, UI).
- **`vault/`** — bóveda de Obsidian con las decisiones de arquitectura (ADRs), el modelo de dominio, referencias de diseño UI/UX y notas de aprendizaje. Ábrela como vault en Obsidian para navegar con enlaces y grafo.

## Cómo retomar el trabajo

1. Abrir `vault/` como bóveda en Obsidian para consultar contexto.
2. Abrir la raíz del repo con Claude Code — leerá `CLAUDE.md` automáticamente.
3. Seguir el estado y próximos pasos listados en `vault/00 - Inicio.md`.
